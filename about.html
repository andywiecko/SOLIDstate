About

SOLIDstate is a C++ library, whose main purpose is to study (relatively small systems up to \( L\sim30 \) strongly correlated systems. SOLIDstate depends on the powerful C++ template library Armadillo. The main feature is that user can easily specify the type of the solving predefined Hamiltonians (sparse/dense), type of the data (double/complex), type of the considering statistical ensemble, and more...
